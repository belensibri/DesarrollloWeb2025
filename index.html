Datos generales del curso

Curso: Desarrollo Web

En este curso indagamos más en el mundo de las páginas web, aprendemos como crearlas, pero sobre todo como hacerlas funcionales e interactivas.

Contenidos: HTML, CSS, JS y la aplicación de estos para una página web funcional

Guías realizadas

Guía 1: Introducción y conceptos básicos de HTML y CSS

Contiene ejercicios para identificar la estructura básica de un documento web, aplicar reglas de estilo CSS a los elementos HTML usando estilos embebidos y hojas de estilo externas, utilizar los selectores CSS básicos y las propiedades CSS para la utilización de fuentes y textos.

Contiene ejercicios para utilizar los diferentes tipos de posicionamiento CSS y diferenciarlos apropiadamente, uilizar las propiedades CSS relacionada con la visualización de elementos, maquetar sitios web con diseños fijos y fluidos

Guía 4: Introducción al Responsive Design

Contiene ejercicios para crear y estilizar formularios utilizando la diversidad de controles de HTML5, utilizar correctamente la meta-etiqueta viewport, defina correctamente las media queries para aplicar reglas de estilo en función del ancho del viewport del navegador, utilice los conocimientos adquiridos durante el transcurso de la asignatura para la construcción de páginas web responsivas.

Guía 5: Introducción a JavaScript

Contiene ejercicios para adquirir dominio de los elementos sintácticos básicos del lenguaje JavaScript, ser capaz de generar código HTML desde secuencias de comando de JavaScript, poder crear páginas web utilizando métodos básicos de interacción con el usuario, utilizar correctamente estructuras selectivas y repetitivas en JavaScript.
